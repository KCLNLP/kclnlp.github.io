Sentiment Analysis
We carry out research in a range of tasks in sentiment analysis, including sentiment classification, emotion ranking, sentiment-topic extraction, Twitter sentiment analysis, dialogue emotion detection, emotion cause extraction, aspect-based sentiment analysis, stance detection and sarcasm detection.
